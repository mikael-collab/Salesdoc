📖 How to set up & deploy this app
Sheet format expected:
• Column A = Field labels (e.g. "Client Name:")
• Column B = Values (editable by salesperson)
• Empty B cell + short A text = section header
• Each sheet tab becomes a navigation tab in the app

Google Cloud setup:
1. Go to console.cloud.google.com
2. Create a project → enable Google Sheets API
3. Credentials → Create OAuth 2.0 Client ID → type: Web Application
4. Add your hosted URL to Authorized JavaScript Origins
5. Copy the Client ID into the field above

Per salesperson setup:
• Share their sheet with their Google account
• They open the app, paste their sheet URL, sign in
• Their settings are saved on the device for next time

Hosting: Host this HTML file on any web server, GitHub Pages, or Netlify. It requires no backend.
